# neuro-receipts-explorer

NeuroMesh defines a humanoid-first intelligence layer on Solana that any robot can install to become software-defined, safety-bounded, and market-connected. This explorer visualises **CTV/A receipts**, agents, and a simplified “Superbrain” network.

## Run

Backend:
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8787
```
